Creating Links with HTML

Within a standard web page, there are several reference points that contain anchors <a>, which are links to other websites and/or pages. Almost all websites today serve more than one purpose, which means they may contain many pages that link to certain functions and information. In this blog post, I will discuss how to create links to both external and internal sources with HTML.

Links to External and Internal Sources

As stated before, we utlize the anchor tag <a> to create links within a page. To do so, we nest the word or image we would like to serve as a link within an anchor tag. After, an href attribute is added to the opening anchor tag. The href attribute specifies the address where the link should go once clicked.

Another optional attribute that can be added to the anchor tag is a target. Specifying a target allows the link to be opened under several options. The chart above, states the common target attributes for an anchor tag and their purposes. So with the following example: CSUN
The link opens in another tab. And this is done so with the following: 

<a href="https://www.csun.edu/" title="CSUN Webpage" target="_blank">CSUN</a>
Another attribute to notice is "title". This global attribute is commonly used to aid with page accessibility. With a title attribute, these can be read and interpreted by assistive devices, search engines, etc.

To link to an internal webpage, the same steps as linking to an external website are taken. The only difference is that, rather than a web address, the href would be directed to another html file.

By Region

Another way to utlize the anchor tag, is by linking to different page regions. This can be commonly found in pages that have navigation buttons located throughout their website. Most common is the "Back to the Top" button that may pop up all the way at a bottom of a page or as you scroll down a website. This can commonly be found on e-commerce websites, long forums, etc.

At the bottom of this page, there is a link that will navigate to the top of this page. To accomplish this with HTML, markers are used throughout the page to signify locational references. For this page, I declared the top to be by the main header of the page. The following is how it is presented: 

<a href="#top" title="Navigate Back to the Top of the Page">Back to the Top</a>
Should you want to specifically link to another internal source at a specific region, the href would have the name of the file followed directly by a # with the location identifier.
